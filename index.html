I didn't get handed a coding curriculum — I found Lua on my own in February 2016 because I wanted to make things that didn't exist yet. I was six. By the time most kids were learning to type I had already broken and rebuilt my first game systems.

Second grade, I was bypassing school network restrictions just to practice building websites. Not to cause problems — I had already learned what the teacher was covering and wanted to keep moving. That same year I started working through algorithms.

Now I co-run Collapse Initiative, Inc. — a game studio with multiple live titles, a clothing brand, and more being built. The work spans game architecture, server systems, web development, cybersecurity, and systems programming.
